Font Awesome Social Media Icons

February 17, 2021 by Rupak Kumar

This blog post shows how to add various social media icons using fontawesome 5.15.2 version. Below are the examples of major social media icons.

You do not need a custom css to do this, in the past 4.7.0, i used a custom css however with the new version you can control the size, design and color using the fontawesome defined css classes. These are pretty handy to use in your website.

Facebook Example Icon

We are going to look at 3 attributes of the icon. They are:

  • Size
  • Shape
  • Color

Size

Example code below :-

<i class="fab fa-facebook-f fa-xs"></i>
<i class="fab fa-facebook-f fa-sm"></i>
<i class="fab fa-facebook-f fa-lg"></i>
<i class="fab fa-facebook-f fa-2x"></i>
<i class="fab fa-facebook-f fa-3x"></i>
<i class="fab fa-facebook-f fa-4x"></i>
<i class="fab fa-facebook-f fa-5x"></i>
<i class="fab fa-facebook-f fa-7x"></i>
<i class="fab fa-facebook-f fa-10x"></i>

Below is how it will appear in your website.

Shape

You can search for the supported shapes of facebook icons in fontawesome here.

You will find 3 types as below:

<i class="fab fa-3x fa-facebook-f"></i>
<i class="fab fa-3x fa-facebook"></i>
<i class="fab fa-3x fa-facebook-square"></i>

Below is how it will appear in your website.

Color

You can use custom color for the icons using the span element. See examples below with size 4x and circle icon.

<span style="color:Mediumslateblue;">
 <i class="fab fa-4x fa-facebook"></i>
</span>
<span style="color:rgb(216, 84, 24);">
 <i class="fab fa-4x fa-facebook"></i>
</span>
<span style="color: rgb(19, 209, 29);">
 <i class="fab fa-4x fa-facebook"></i>
</span>
