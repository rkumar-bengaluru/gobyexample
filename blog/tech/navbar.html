Responsive Navigation Bar

February 24, 2021 by Rupak Kumar

Before we start let's make sure you see the demo below if that is what technically you are looking for. There are thousands are web resources around this topic and they talk about one aspect or the other, the demo will help save your time, if this is not what you are looking then happy web sourcing otherwise continue reading. By the end of this blog you will be able to write your own responsive bar as below.


Now that you are reading this means this is what you want. Word of caution - do not mix too many web frameworks in your site that just kills the whole purpose. With the ocean of frameworks around with so many over-lapping classes that you will end up debugging most of your time on why the look and feel do not matches with the defined description.

To be very precise i have used bootstrap v 4.6.0 for getting this responsive menu. Do not mix with other frameworks like materialize ui etc. The minimal you need are below:-

What Next??? Open Trigger

So far we have defined the div which has the navigation definition meaning what menus and submenus required for the project. When the page loads this div width is set to 0.

<style>
 .sitenav {
  width: 0;
 }
</style>

Now let's code the trigger button which onClick will set some width to sitenavigation div.

<div class="col-md-auto">
 <button type="button" class="btn btn-primary" onClick="openNav()">
  <i class="fa fa-bars"> Demo</i>
 </button>
</div>

Check the above code, when the button is clicked a javascript function is called which will set the width as below:

<script>
 function openNav() {
   document.getElementById("sitenavigation").style.width = "400px";
 }
</script>

Let's code Close Trigger

<script>
 function closeNav() {
   document.getElementById("sitenavigation").style.width = 0;
 }
</script>

Finally Let's Have a close button.

The close trigger has to be part of the sitenavigation div only. Let's add the close button on the sitenavigation div.

<div> class="container">
 <div id="sitenavigation" class="sitenav">
  <button onClick="closeNav()" type="button" class="btn btn-primary m-2">
   <i class="fas fa-times"></i>
  <button>
  ...menus and submenus follows
 </div>
</div>
